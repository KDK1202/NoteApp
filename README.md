# Notes App
A simple desktop notes application built with Python.
It allows users to create, view, search, rename, and delete notes through a clean graphical interface.

This project was built to practise Python GUI development, file handling, and basic application logic.

Features:
- Create new notes
- View saved notes
- Search notes by name
- Rename existing notes
- Delete notes
- Notes are saved locally on the userâ€™s device

Technologies used:
- Python
- Tkinter (GUI)
- File handling

How to run:
1. Install Python (if not installed)
2. Download or clone this repository
3. Navigate to the project folder
4. Run the python file: notes.py

File stucture:
- /notes (folder that saves notes)
- notes.py

Purpose of this project:
This project was created as a learning exercise to:
- Improve Python programming skills
- Learn how to build desktop applications
- Understand basic UI logic and user interaction
